Suggest the use of an adblocker if reader is not using one.

== Description ==

Using a widget it suggest to use an adblocker (uBlock Origin) if a reader is not using one.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/please-adblock` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add the widget using the  Appareance->Widgets screen.

== Frequently Asked Questions ==

= Why? =

Because adblockers are good. They not only give you a nicer, faster and less capitalist browsing experience, but they also take a huge role in securing and respecting your privacy.

= Why uBlock Origin and not X/Y/Z? =

It is the nicer one (Dec 2015)
